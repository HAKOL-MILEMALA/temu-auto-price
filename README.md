# 💰 התאמת מחיר אוטומטית בטמו (Temu)

סקריפט אוטומציה חכם ויעיל לתוספת הדפדפן **Tampermonkey**, המיועד לבצע סריקה, הגשה ואיסוף אוטומטי של החזרי כספים בעקבות ירידת מחירים (Price Adjustment) באתר Temu.

---

## ✨ תכונות עיקריות

* **🔄 זיהוי ואוטומציה מלאה:** לוחץ אוטומטית על כפתורי "התאמת מחיר", בחירת זיכוי מהיר ושליחת הטופס ללא צורך במגע יד אדם.
* **🛡️ בועת בקרה מודרנית:** מציגה בכל רגע נתון את מצב התהליך (טעינה, הרצה, הצלחה או סירוב) במיקום מרכזי בראש העמוד.
* **🛑 כפתור עצירה מיידי:** מאפשר לך לעצור את האוטומציה בכל שלב בלחיצת כפתור אחת. הסקריפט מציג הודעת עצירה ונעלם באלגנטיות לאחר 2 שניות.
* **⏱️ השהייה וספירה לאחור:** בעמוד ההזמנה מבוצעת השהיה של 5 שניות, ובשלב הסגירה מופיעה ספירה לאחור של 3 שניות כדי להעניק לך חלון זמן לעצור את התהליך.
* **🚪 סגירת עמוד חכמה:** סוגר את העמוד אוטומטית גם במידה ומופיעה חלונית הסירוב ("מצטערים, אין באפשרותך...") וגם עם הגעה לעמוד סיכום ההחזר המוצלח.

---

## 🚀 הוראות התקנה

1. ודא שמותקן אצלך תוסף **Tampermonkey** בדפדפן (Chrome / Edge / Firefox).
2. לחץ על כפתור ההתקנה המהירה כאן למטה:
   
   👉 **[לחץ כאן להתקנת הסקריפט](https://raw.githubusercontent.com/HAKOL-MILEMALA/temu-auto-price/main/%D7%94%D7%AA%D7%90%D7%9E%D7%AA%20%D7%9E%D7%97%D7%99%D7%A8%20%D7%90%D7%95%D7%98%D7%95%D7%9E%D7%98%D7%99.user.js)**

3. ייפתח חלון של Tampermonkey — לחץ על **התקן / Install** (או Update אם מותקנת גרסה קודמת).
4. זהו! מעכשיו הסקריפט יתעדכן אוטומטית בכל פעם שישוחרר עדכון חדש בגיטהאב.

---

## 🛠️ כיצד להשתמש?

1. היכנס לעמוד פרטי ההזמנה שלך ב-Temu.
2. בראש המסך תופיע בועת הסטטוס המעוצבת של הסקריפט.
3. הסקריפט ימתין 5 שניות (בזמן הזה ניתן ללחוץ על עצירה) ואז יבדוק אם כפתור התאמת המחיר קיים וילחץ עליו.
4. אם ישנו החזר זמין, הוא יבצע את כל שלבי האישור בשבילך.
5. במידה ואתה מעוניין להפסיק את הריצה ולבחון את העמוד בעצמך, פשוט לחץ על הכפתור **"עצור אוטומציה והישאר בעמוד"** בבועה העליונה.
